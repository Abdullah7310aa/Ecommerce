# My Portfolio Web Project

## Overview
This project is a **dynamic and modular portfolio webpage** built using a **Drag & Drop Modules system**. Each module can be easily customized through **fields** without touching HTML/CSS code. The project uses a **Dark Mode design** for a modern, sleek look.

---

## Modules

### 1. Hero Module
- **Fields**:
  - `Name` → Your name
  - `Title` → Your profession/title
  - `Description` → Short bio or introduction
  - `Image` → Profile image
- **Features**:
  - Responsive layout
  - Dark Mode
  - Image hover effect

### 2. Skills Module
- **Fields**:
  - `skill_1` to `skill_6` → Your top skills
- **Features**:
  - Styled skill list with hover effects
  - Flex layout for responsiveness

### 3. Projects Module
- **Fields**:
  - `project1_name`, `project1_desc`
  - `project2_name`, `project2_desc`
  - `project3_name`, `project3_desc`
- **Features**:
  - Dark-themed project cards
  - Hover effects for interaction

---

## Features
- **Drag & Drop Modules**: Easily add, remove, or reorder sections on the page.
- **Dynamic Fields**: Update text or images without changing the HTML.
- **Dark Mode**: Sleek, modern design suitable for portfolios.
- **Responsive Design**: Works on desktop, tablet, and mobile screens.
- **CSS Hover Effects**: Subtle animations for projects and images.
